# Release Notes Q1 2026

## Overview

This document summarizes the key updates, improvements, and bug fixes released in Q1 2026 for the Atea Managed Public Cloud.

## New Features

- **Feature 1:** New Azure Health Dashboard available through the governance service.
- **Feature 2:** Automatic test implemented for all services except governance.

## Improvements

- **Improvement 1:** Clarification of service scope for restore in capability description for Managed Public Cloud VM operations document.

## Bug Fixes

- **Bug Fix 1:** Various bug fixes related to V2 development.

## Known Issues

- **Service health alerts:** There is a known issue with the Terraform Provider that prevents us from changing the priority of health alerts.
- **Auto-resolving health alerts:** Health alerts do not auto-resolve, making it harder to maintain accurate status information in our dashboards.
